# market-pulse-signals · 个股信号面板(静态页)

「市场脉搏」研究系统的个股关注池信号面板——每个交易日盘后由脚本自动生成并推送到本仓库, 经 GitHub Pages 呈现。

- **是什么**: 关注池(~30只A股, 行业代表+自选)的技术面/估值/相对强弱数据面板与多空信号底稿, 以及(后续)信号历史命中率统计。
- **怎么看**: `index.html` 池子总览 → 点单股进 `s/<code>.html` 详情页。
- **数据来源**: akshare(公开行情/财务数据), 每日盘后更新; 数据可能有缺失或延迟, 以交易所披露为准。
- **免责声明**: 本仓库全部内容由程序自动生成, 属研究性质的个人数据整理, **不构成任何投资建议**, 不含买卖推荐/目标价; 提及的公司仅为行业观察锚点。据此操作风险自负。

(生成端: 私有仓库 market-pulse 的 publish_pages 管线; 本仓库只存放生成的静态页。)
